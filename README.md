# 機能一覧
* ユーザー登録、編集  
* 各ユーザーの記事作成、編集、削除機能（編集、削除は本人のみ）  
* ユーザーフォロー機能  
* 記事のイイね機能  
* 記事をカテゴリ分類  
* ユーザー間の非同期チャット機能  

# 制作環境
* Ruby 2.5.1  
* Ruby on Rails 5.2.3  
* HAML, SCSS, JavaScript(ライブラリ: Jquery)  
* Heroku  
* AWS(S3)  
本番環境はHerokuを使用。画像アップロードはgem'CarrierWave'を使用し、S3へ保存。
